# Operating Systems and You – Becoming a Power User

Introduction

This assessment demonstrates practical knowledge of Windows operating system administration and power user skills. The task focuses on understanding operating system internals, managing users and groups, working with the NTFS file system, and installing software using command-line tools.

Using PowerShell and built-in Windows utilities, system information was gathered, new users were created and assigned to groups, file permissions were configured, and software was installed through a command-line package manager. These tasks highlight essential technical support and system administration competencies required in real-world IT environments.
